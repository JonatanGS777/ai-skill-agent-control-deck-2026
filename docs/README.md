# Docs Portal

Este directorio concentra documentacion operativa del repositorio.

- Portal HTML generado: `docs/portal/index.html`
- Guia de inicio: `docs/quickstart.md`
- Patrones: `docs/patterns.md`
- Anti-patrones: `docs/anti-patterns.md`
- Ejemplos top: `docs/examples-top.md`
- Gobernanza: `docs/governance/*`

Para regenerar el portal:

```bash
make docs-portal
```
